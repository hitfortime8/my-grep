# Тестовое задание : `Аналог команды grep`
---
### Использование

Программа используется в следующем формате: `python grep.py <options> <word> <path>`

### Реализованные флаги

```
-i - Поиск без учета регистра
-n - Отображение номера строки, в которой найдено слово
-v - Инвертированный поиск (искать строки, где слово не встречается)
-c - Подсчет строк, в которых встречается слово. Не выводит саму строку
-r - Поиск рекурсивно по директории, включая вложенные папки
-l - Выводит только имена файлов, в которых найдены совпадения
-h - Выводит подсказку по использованию программы
```
**Флаги -l, -c убирают строки из вывода**

### Позиционные аргументы

Опции `word` и `path` являются позиционными и обязательными, без них программа не запустится
